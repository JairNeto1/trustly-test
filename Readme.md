# Shoes.on

**The best Sneakers in town**

That's an online shoe store that integrates with Trustly's PayWithMyBank mechanism.

![](/public/print.png)

### Installation

Run it on your machine:

```
  yarn install
```

then:

```
  yarn start
```

or simply visit the link:
[Shoes.on](https://shoeson.herokuapp.com/)
